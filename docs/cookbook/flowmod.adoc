=== FlowMod Messages


==== Delete Flows in All Tables

[source,yaml]
----
type: FLOW_MOD
msg:
  command: DELETE
  table_id: ALL
----


==== Delete Flows in a Table 1

[source,yaml]
----
type: FLOW_MOD
msg:
  command: DELETE
  table_id: 1
----


==== Add a Table Miss Flow Entry to Send Packets to Controller

[source,yaml]
----
type: FLOW_MOD
msg:
  command: ADD
  table_id: 0
  instructions:
    - instruction: APPLY_ACTIONS
      actions:
        - action: OUTPUT
          port: CONTROLLER
          max_len: MAX
----

==== Add a Table Miss Flow Entry to Drop Packets

[source,yaml]
----
type: FLOW_MOD
msg:
  command: ADD
  table_id: 0
----
